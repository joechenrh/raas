What is Triage CI? dont-retest skill ↗
When a PR enters ci-failed, clicking Triage CI invokes Codex with the dont-retest skill. It collects every failing CI check, extracts failure signatures from logs, classifies the root cause, and posts a structured report as a PR comment. Below is an example of the report it produces:
