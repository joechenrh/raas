What is Triage CI? dont-retest skill ↗
When a PR enters ci-failed, clicking Triage CI invokes Codex with the dont-retest skill. It collects every failing CI check, extracts failure signatures from logs, classifies the root cause, and stores a structured report in the dashboard for hover inspection.
